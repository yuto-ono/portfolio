About

本サイトをご覧いただき、ありがとうございます。Webエンジニアの小野優人（おのゆうと）と申します。1995年生まれ、福岡県出身の福岡県在住です。現時点（2019年6月）で24歳です。

高校卒業後、しばらくアパレルショップのECサイトの管理・発送業務等のアルバイトをしていましたが、2016年9月、エンジニアをしている友人の紹介で、ソフトウェア開発を行うIT企業に入社することができました。 ほぼ業界未経験の僕を採用してくださったことに今でも感謝しています。 その会社では、Web制作の受託案件をこなしながら、自社開発の案件にも携わることができました。 Web系の案件では、 HTML / CSS / JavaScript といった基本的なスキルはもちろん、 WordPress などの CMS 組み込み、 Vue.js や Laravel などのフレームワークを使った開発のスキルも身に着けることができました。 自社開発のプロジェクトでは、 C# / SQLServer を使った業務アプリケーションの開発や、 Laravel を使ったポイントシステムの開発をなど行い、様々なスキルを身に着けることができました。

それから2年以上たって、2018年12月、東京にあるWeb制作会社に転職しました。福岡県に住んだまま、フルリモートで働かせていただいております。本当に感謝です。 そこでWeb関連のスキルがさらに鍛えられています。そしてついに、2019年10月からフリーランスエンジニアとして活動させていただくことになりました。

この仕事をやってて思ったのが、めっちゃ楽しいです。やはり、一からWebサイトやWebサービスを作るのはワクワクしますし、やりがいがありますね。 こんな私がお役に立てることがございましたら、ぜひご一緒にお仕事ができれば幸いでございます。
